# Basketball R3F MVP

Mini jogo de basquete feito com React Three Fiber + Rapier.

## Rodar

```bash
npm install
npm run dev
```

Abra o endereço que aparecer no terminal, normalmente `http://localhost:5173`.

## Controles

- A/D ou setas: ajustar mira lateral
- Segurar Espaço: carregar força
- Soltar Espaço: arremessar
- R: resetar bola
- No celular: use os botões na tela
